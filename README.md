
# LintCustomisation

**Author:** Yiyi Wang
*(Please indicate the source code when forwarding)*

This project implements a custom Lint Detector Generator. In large-scale software development projects, this tool successfully achieved automated scanning of Java class names, function names, and illegal calling statements. Additionally, a batch Lint rule generation script was written, significantly enhancing the efficiency of the development team's code standard review.

## File Descriptions

* **`build.gradle`**
    * Provides guidance on how to write your `build.gradle` file in Android Studio for this project.

* **`CreateCodeDetector.java`**
    * Implements the functionality to automatically generate a Method Detector.

* **`ExampleMethodDetector.java`**
    * An example of a Detector that was generated by `CreateCodeDetector.java`.

* **`CreateRegistry.java`**
    * Implements the functionality to automatically generate the Issue Registry.

* **`CodeReTemp.java`**
    * A custom script to modify the format of your detected Java files. This makes it easier to get the Java files in the project and generate the necessary code.
    * **Note:** Its functionality depends on the naming conventions of your Java files.

## LintCustomisation - 自定义 Lint 探测器生成器

> 原作者：Yiyi Wang
> 转载请注明出处。

本项目是一个用于自动生成 Android Lint 探测器（Detector）的工具。在大型软件开发项目中，该工具成功实现了对 Java 类名、函数命名及非法调用语句的自动化扫描。此外，还编写了批量 Lint 规则生成脚本，显著提高了开发团队的代码规范审查效率。

---

### 核心文件说明

* **`build.gradle`**
    * 提供了关于如何在 Android Studio 中配置 `build.gradle` 文件的说明。

* **`CreateCodeDetector.java`**
    * 用于自动生成“方法探测器”（Method Detector）的实现。

* **`ExampleMethodDetector.java`**
    * 一个由 `CreateCodeDetector.java` 所生成的探测器（Detector）的示例代码。

* **`CreateRegistry.java`**
    * 用于自动生成“问题注册表”（Issue Registry）的实现。

* **`CodeReTemp.java`**
    * 一个自定义代码脚本，用于修改项目中已检测到的 Java 文件的格式。
    * 这使得抓取项目中的 Java 文件并生成相应代码变得更加容易。
    * **注意**：此脚本的运行效果取决于你项目中 Java 文件的命名规范。
